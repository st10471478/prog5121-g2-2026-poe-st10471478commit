## Project Description

This is the final part of the Chat Application that completes all requirements for PROG5121. The application combines Part 1 (Registration/Login), Part 2 (Message Sending), and Part 3 (Arrays, Reports, and JSON File Reading) into one complete system.

### Part 3 Features Implemented

| Feature | Description | Status |
|---------|-------------|--------|
| Parallel Arrays | SentMessages, DisregardedMessages, StoredMessages, MessageHash, MessageID | Complete |
| Display Stored Messages | Shows sender and recipient of all stored messages | Complete |
| Longest Message | Finds and displays the longest stored message | Complete |
| Search by Message ID | Searches arrays and displays recipient and message | Complete |
| Search by Recipient | Finds all messages for a particular recipient | Complete |
| Delete by Hash | Removes message from arrays using hash | Complete |
| Full Report | Displays all stored message details with totals | Complete |
| JSON File Read | Reads stored_messages.json into array | Complete |
| JUnit Tests | 12 tests covering all Part 3 functionality | Complete |
