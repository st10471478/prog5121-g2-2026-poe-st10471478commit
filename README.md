## Project Description

This console application implements a user registration and login system for a Chat Application. The system validates user credentials including username format, password complexity, and South African cell phone numbers with international country codes.

### Features Implemented

| Feature | Description | Status |
|---------|-------------|--------|
| Username Validation | Checks for underscore (_) and maximum 5 characters |  Complete |
| Password Complexity | Validates 8+ characters, capital letter, number, special character |  Complete |
| Cell Phone Validation | Regex-based validation for +27 format |  Complete |
| User Registration | Stores validated user credentials |  Complete |
| Login Authentication | Verifies credentials against stored data |  Complete |
| Unit Testing | JUnit 5 tests for all validation methods |  Complete |
